# 📈 Quant Finance Project

Analysis of ONGC stock during the Iran-USA geopolitical tensions (2025-2026)

## What this project covers
- ONGC vs Crude Oil price correlation
- SMA20 & SMA50 crossover strategy
- 30-day rolling correlation analysis
- Event-driven analysis backed by real news

## Tools Used
- Python, Jupyter Notebook
- yfinance, pandas, numpy, matplotlib

## Key Finding
ONGC only strongly follows crude oil during extreme geopolitical events.
In normal times, domestic factors dominate.
